# Haifeng-2026-Wuhu-Departure

> 芜宣机场 (WHA) 直飞航线辐射图 — 国内 22 城，3 个梯队分级显示
> Wuhu Xuanzhou Airport (WHA) — radiating hub-and-spoke map of 22 direct domestic destinations, grouped into 3 tiers

## 这是什么

一张 Leaflet 地图：
- 中心枢纽：**芜宣机场** (WHA / WHU_XY, 31.104, 118.667)
- 22 条辐射航线：从枢纽指向每个直飞国内目的地
- 三档分级（颜色 + 线宽）：

| 梯队 | 含义 | 颜色 | 数量 |
|------|------|------|------|
| **Tier 1** | 每日或近每日 / 主力航线 | 实线 `#2b6cb0` | 10 城 |
| **Tier 2** | 每周 3-7 班 / 含经停 / 季节性加密 | 实线 `#5a8ece` | 8 城 |
| **Tier 3** | 稀疏 / 暑期加密 / 历史通航 | 虚线 `#a0aec0` | 4 城 |

## 交互

- **鼠标悬停航线** → 弹出 tooltip（航司 + 班次 + 时长 + 梯队标）
- **点击航线** → 完整弹出框（含城市介绍、备注、经停提示、数据来源）
- **点击枢纽圆点** → 机场详情（开放日期、机场等级、距离湾沚）
- **点击左侧城市卡片** → 地图飞越到该城市

## 与 `Ivan-2026-China-Travel` 的区别

| 项 | Ivan | 本项目 |
|----|------|--------|
| 行程类型 | 时间顺序旅行（15 段 LEGS） | 静态辐射图（22 条航线） |
| 数据结构 | `LEGS + ORDER + HSR_ROUTES` | `ROUTES + TIER_INFO` |
| 标记 | 城市编号 1-9 + 交通节点 | 中心枢纽 + 目的地 22 点 |
| 视觉流程 | 起点→终点逐段 | 中心→全方位辐射 |
| 适用场景 | 一次旅行的 10-30 个时段 | 一个机场可达目的地全集 |

**没有 LEGS/ORDER/HSR_ROUTES 标准结构**——这个项目是 hub-and-spoke 模型，与时间顺序旅行本质不同。`validate.sh` 不会识别 ROUTES，所以没跑那个脚本，但 JS 语法已通过 `node -c` 验证。

## 数据来源

| 文档 | 时间 | 内容 | 备注 |
|------|------|------|------|
| 2025-2026 冬春航季公告 | 2025-10-18 | 20 城市、24 航线 | 芜湖市政府 + 腾讯新闻原文 |
| 2025 夏秋季航季公告 | 2025-03-29 | 13 城市、16 航线 | 春秋换季原创报道 |
| FlightConnections WHA | 2026-06-03 更新 | 当前 11 城市、~70 航班/周 | 国际航线数据网站 |

**强时效提示**：航班班次随航季调整（每年 3 月底 + 10 月底两次换季）。订票前请确认：
1. 芜宣机场官网（whxc-airport.com）公布时刻
2. 航旅纵横 / 携程实时查询
3. FlightConnections 更新日期

## 已知限制

1. **航司信息不完整**——有些航线只列「—」如西安/昆明等历史通航
2. **经停航线的航班号**未一一列出（如 ZUH/HRB 是 GJ8709/8710 同一航班经停）
3. **2026 夏秋季的具体班次公告**暂无原文报道，目前以 2025 夏秋季公告 + FlightConnections 2026-06 实时数据合成
4. **航线坐标精度**——目的地坐标取**机场位置**（不是市中心），与 Leaflet + OSM 瓦片兼容
